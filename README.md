# פרויקט לוח מחוונים לאבטחת ענן (Cloud Security Dashboard)

ברוכים הבאים לפרויקט הלימודי שלכם!
פרויקט זה מדמה מערכת דשבורד (לוח מחוונים) עבור ניטור אבטחת ענן. המערכת נבנתה באמצעות React, Tailwind CSS ורכיבי UI מודרניים.

## 🎯 מטרת הפרויקט
המטרה שלכם היא להכיר את מבנה הפרויקט, להבין איך קומפוננטות (Components) מתקשרות אחת עם השנייה, ולבצע שינויים ושיפורים בקוד הקיים.

## 🚀 איך מתחילים?

1.  **התקנת ספריות**:
    פתחו את הטרמינל בתיקיית הפרויקט והריצו:
    ```bash
    npm install
    ```

2.  **הפעלת הפרויקט**:
    כדי לראות את האתר רץ מקומית במחשב שלכם:
    ```bash
    npm run dev
    ```
    לאחר מכן, פתחו את הדפדפן בכתובת שמופיעה (בדרך כלל `http://localhost:5173` או דומה).

## 📂 מבנה הפרויקט
הקבצים החשובים נמצאים בתיקיית `src`:
*   `src/components`: כאן נמצאים כל הרכיבים של האתר (כפתורים, גרפים, תפריטים).
*   `src/components/ui`: ספריית רכיבים בסיסיים (כמו כפתורים ותיבות טקסט).
*   `src/Layout.tsx`: המבנה הכללי של העמוד.
*   `src/App.tsx`: נקודת הכניסה הראשית של האפליקציה.

## 📝 משימות לתלמידים

בחרו את המשימות לפי רמת הקושי המתאימה לכם. מומלץ להתחיל מהקלות ולהתקדם.

### רמה 1: מתחילים (היכרות עם הקוד)
1.  **שינוי כותרת**: אתרו את הכותרת הראשית בדשבורד ושנו אותה ל-"Radware Student Dashboard".
2.  **עדכון Footer**: מצאו את רכיב ה-`Footer` והוסיפו את השם שלכם ליד זכויות היוצרים.
3.  **שינוי צבע**: נסו לשנות את צבע הרקע של אחד הכרטיסים (Cards) לצבע אחר לבחירתכם.

### רמה 2: מתקדמים (הוספת פונקציונליות)
1.  **כפתור חדש**: הוסיפו כפתור "הורדת דוח" (Export Report) ליד הכותרת הראשית. כרגע הוא יכול רק להקפיץ הודעה (`alert`) כלוחצים עליו.
2.  **רכיב חדש**: צרו קומפוננטה חדשה בשם `StatusCard.tsx` שמציגה סטטוס "System Healthy" עם אייקון ירוק, והוסיפו אותה לדשבורד הראשי.
3.  **תיקון "באג"**: בדקו את הניווט (Navigation). האם כל הלינקים עובדים? נסו לגרום לאחד הלינקים להוביל לעמוד או להציג הודעה אחרת.

### רמה 3: מומחים (עבודה עם מידע ולוגיקה)
1.  **מידע דינמי**: בגרף התעבורה (`TrafficChart`), נסו לשנות את הנתונים הקבועים (Hardcoded) למערך נתונים שאתם מגדירים במשתנה נפרד או מקבלים כ-Props.
2.  **Dark Mode**: בדקו איך המערכת מתנהגת ב-Dark Mode ונסו לשפר נראות של רכיב שלא נראה טוב ברקע כהה.
3.  **סינון**: הוסיפו תיבת חיפוש (Input) שמאפשרת לסנן רשימה כלשהי בדף (אם קיימת) או פשוט מדפיסה לקונסול את מה שהמשתמש מקליד בזמן אמת.
